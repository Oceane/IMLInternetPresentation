Prelude:

The world wide web was invented by Tim Berners-Lee in 1989. He also established the first web server, web browser/editor, HTTP and HTML between 1989 and 1992. On August, 6th 1991, he launched the very first website in history (info.cern.ch).

The internet in 1992

1: Berners-Lee distributes a free web browser that allows access to 26 web servers; no pictures just links
2: Example of an HTML page in 1992 (http://www.w3.org/History/19921103-hypertext/hypertext/WWW/MarkUp/MarkUp.html)
3: 50 Websites on 26 servers
4: “surfing the web” is coined
5: first photo posted to web (http://www.telegraph.co.uk/technology/news/9391110/How-the-first-photo-was-posted-on-the-Web-20-years-ago.html)

The internet in 1993

1: World Wide Web enters the public domain
2: Marc Andreesen creates Mosaic, the first “user-friendly” browser that displayed graphics and pictures (insert picture of mosaic)
3: Mosaic influences a spike in users on the web (~14 million)
4: notable quotes: "'Net-surf'...that's what the cool kids are calling it."

The internet in 1994

1: World Wide Web Consortium (W3C) created
2: Mosaic becomes Netscape Navigator
3: CSS proposed by Håkon Wium Lie
4: Geocities created
5: Microsoft publishes their first website
6: Yahoo is launched
7: Majority of internet users use dial-up (pay-per-minute) services like AOL
----->a: fax machine more popular than email because email was too expensive

The internet in 1995

1: Javascript, Java and PHP released
2: HTML 2.0 released
3: Microsoft Internet Explorer 1.0 and 2.0 released; free on all Windows devices
4: Theglobe.com created; first social networking site
5: Amazon launched
6: AuctionWeb (Ebay) launched

The internet in 1996

1: More emails sent through web, than letters sent through post office
2: eCommerce, and dot-com companies start to generate interest and revenue
3: Browser war between Microsoft and Netscape begin
------>a: JScript (Microsofts response to Javascript) first appears in Internet Explorer 3.0, which also support some CSS
4: Nokia releases the Nokia 9000 Communicator, the first cell-phone with internet capabilities.
5: first viral video of dancing baby (https://www.youtube.com/watch?v=-5x5OXfe9KY)
6: HoTMail launches

The internet in 1997

1: Internet Explorer 4.0 released, integrated into Windows software
----->a: Microsoft has the financial support, and market support in browser war

The internet in 1998

1: Open Diary, first blogging platform that encourages users to comment and interact with the publisher (hint of Web 2.0)

The internet in 1999

1: Paypal launched
2: Napster launched

The internet in 2000

1: Microsoft has beat Netscape in the browser wars
2: More than half of American homes have Internet access
3: dot-com bubble peaks and then crashes
4: Google becomes largest search engine
5: XHTML released
